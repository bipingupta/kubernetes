---------------------------------------------------------------------------
                     START Docker Compose
---------------------------------------------------------------------------
Execute the following command to pull the images and create containers:
docker-compose up -d

Verify both are running:
docker ps

---------------------------------------------------------------------------
                     STOP KAFKA BROKER
---------------------------------------------------------------------------
docker-compose down
